## 一、项目概况

| 项目 | 详情 |
|------|------|
| 主程序 | `claude_floating_launcher.py` (781 行) |
| 技术栈 | Python 3.12 + PyQt5 + PyInstaller |

## 三、安全漏洞

### 🔴 Critical (1)

| ID | 漏洞 | 影响文件 | 行号 |
|----|------|----------|------|
| **S-01** | **PowerShell 命令注入** — f-string 拼接路径未转义。用户可选的自定义安装路径含 `$`、`` ` ``、`"` 等字符时可注入任意 PowerShell 代码 | `claude_floating_launcher.py`、`installer/install.py`、`installer/setup_wizard_tk.py`、`install_startup.py` 等 7 处 | 各处 PowerShell 调用 |

### 🟠 High (2)

| ID | 漏洞 | 影响文件 | 行号 |
|----|------|----------|------|
| **S-02** | **`--dangerously-skip-permissions` 无安全警告** — 当前配置已启用 skip-permissions + 开机自启，Claude Code 启动即获完全系统权限。设置界面仅显示为普通单选框 | `claude_floating_launcher.py` | 124, 187-206 |
| **S-03** | **`launcher.vbs` 含硬编码绝对路径** — 泄露用户真实姓名和完整目录结构，且该文件被提交到版本控制 | `launcher.vbs` | 2 |

### 🟡 Medium (4)

| ID | 漏洞 | 影响文件 | 行号 |
|----|------|----------|------|
| **S-04** | **BAT 批处理命令注入** — `uninstall.bat` 中 `install_dir` 未转义嵌入，含 `&`、`|` 等字符可注入 | `installer/install.py`、`installer/setup_wizard_tk.py` | 48-87, 40-74 |
| **S-05** | **配置明文存储敏感路径** — `launcher_config.json` 存于脚本目录而非 `%APPDATA%`，含工作目录路径和启动模式 | `claude_floating_launcher.py` | 53 |
| **S-06** | **TOCTOU 竞态条件** — `os.path.exists()` 检查后紧跟文件操作，检查与使用之间存在时间窗口 | 多处 | 多处 |
| **S-07** | **TOCTOU in working directory** — 工作目录有效性检查与 `subprocess.Popen` 之间存在竞态，可被符号链接利用 | `claude_floating_launcher.py` | 127-131 |

### 🟢 Low (5)

| ID | 漏洞 | 说明 |
|----|------|------|
| S-08 | DLL 搜索顺序 | `ctypes.windll` 可能从当前目录加载 DLL（shell32.dll 受 Known DLLs 保护，实际风险低） |

## 五、架构评估

### 当前架构问题

| 问题 | 说明 |
|------|------|
| 🔀 **单文件 780 行** | UI、渲染引擎、配置层混在一起，维护困难 |
| ⚡ **100ms 轮询浪费** | 隐藏后仍每秒 10 次检测鼠标位置，影响笔记本电池 |
| 🎨 **每帧重建渐变** | `paintEvent` 中每次创建 7 个新渐变对象 + `QPainterPath` |
| 📦 **53MB 体积** | 未排除 WebEngine/Multimedia/Sql/Network 等未用模块（可缩减至 ~30MB） |
| ⚙️ **四套安装机制** | CLI、tkinter GUI、Inno Setup、独立脚本并存无指引 |
| 💾 **配置位置不当** | 存脚本目录而非 `%APPDATA%` |
| 🖥️ **多显示器支持差** | 断开外接显示器后窗口可能消失在屏幕外 |

### 重复代码

`installer/install.py` 与 `installer/setup_wizard_tk.py` 中以下函数几乎完全重复：
- `create_shortcut()` 
- `create_uninstall_bat()`
- `register_uninstall()`
- `unregister_uninstall()`

## 八、跨平台兼容性说明

本应用程序为 **Windows 专用**。以下为非 Windows 平台不存在或行为不同的功能：

| 功能 | 依赖 | 替代方案 |
|------|------|----------|
| 系统托盘 | `QSystemTrayIcon` | macOS 基本可用，Linux 需检查 |
| `ctypes.windll` | Windows API | 需 `#ifdef` 守卫 |
| `subprocess.CREATE_NO_WINDOW` | Windows 特有 flag | 非 Windows 忽略 |
| Windows Terminal (`wt`) | Windows 10+ | 用系统默认终端 |
| 启动文件夹 | `%APPDATA%\...\Startup` | 需各平台自启方案 |
| `taskkill` / `powershell` | Windows 专有 | 需替代 |
